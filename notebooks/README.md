# Notebooks Overview

This folder contains exploratory and experimental notebooks used during the project.

## Contents

- `eda-fraud-data.ipynb`  
  Exploratory analysis of the Fraud_Data dataset.

- `eda-creditcard.ipynb`  
  Exploratory analysis of the Credit Card fraud dataset.

- `feature-engineering.ipynb`  
  Feature creation and transformation experiments.

- `modeling.ipynb`  
  Baseline and ensemble model experimentation.

- `shap-explainability.ipynb`  
  Model explainability using SHAP.

> Final, reproducible implementations are available in the `src/` directory.
